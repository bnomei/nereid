# Requirements — 08-query-engine

This spec implements typed “ask graph” primitives for agents and UI (reachability, paths, cycles, traces, and session routes).

Normative protocol reference: `docs/protocol-01.md`

## Requirements (EARS)

- THE SYSTEM SHALL answer core query primitives without requiring full AST/ASCII dumps.
- THE SYSTEM SHALL support cross-diagram routes via XRefs and a derived session meta-graph.
